# Free V2Ray Configs

This repository provides a regularly updated collection of V2Ray configurations gathered from public sources.

### Supported Protocols

* VMess
* VLESS
* Trojan
* ShadowsocksR

### Recommended Clients

**Android**

* v2rayNG

**iOS**

* Streisand
* Fair VPN

**Windows / Linux**

* Nekoray
* Hiddify Next
* v2rayN
